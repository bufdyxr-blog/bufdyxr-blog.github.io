这是一个使用 Markdown 格式编写的文档示例。Markdown 是一种轻量级的标记语言，用于编写易于阅读和编写的纯文本格式文档。

简介

Markdown 的语法非常简单直观，易于学习和使用。下面是一些常用的 Markdown 语法元素。

标题

使用 # 符号来定义标题，# 的数量表示标题的级别。

段落

段落由一行或多行文本组成，每段之间留一个空行。

这是第一段的内容。
这是第二段的内容。

链接

链接可以这样创建：[链接文字](https://example.com)。

访问我的网站

图片

插入图片也很简单：![替代文字](图片URL)。

列表

有序列表

  1. 第一项
  2. 第二项
  3. 第三项

无序列表

  • 第一项
  • 第二项
  • 第三项

代码块

对于代码块，可以使用三个反引号（```）包围起来：

或者使用行内样式：

print("Hello, world!")

表格

列1 列2 列3
行1数据1 行1数据2 行1数据3
行2数据1 行2数据2 行2数据3

引用

这是一个引用块。

任务列表

  • 待办事项 1
  • 已完成事项 2
  • 待办事项 3

分割线

要添加一条分割线，可以在一行中输入三个星号（***）或减号（—）。
